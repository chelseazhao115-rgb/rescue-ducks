# Rescue Duck Audio Placeholders

These files are replacement points for mastered audio assets:

- `home-theme.mp3`
- `gameplay-loop.mp3`
- `storm-layer.mp3`
- `intro-ambient.mp3`
- `victory.mp3`
- `failure.mp3`
- `sfx/*.mp3`

The current game uses the procedural Web Audio soundtrack in `src/lib/audio/AudioManager.ts`.
The `.mp3` files here are empty placeholders until final music and SFX are exported.
